# Depósito 1700 - Banco de Imagens

Esta pasta contém o banco de dados de imagens dos itens do depósito 1700 da empresa.

## Estrutura de Pastas

*   **440_Volare:** Imagens dos itens referentes à linha de produção atual 440 Volare.
*   **446_THUNDER:** Espaço reservado para a futura linha de produção 446 THUNDER.

## Padrão de Nomenclatura
As imagens devem seguir o padrão `codigo.png`.

## Observações
Para o upload massivo de imagens (> 1.300 itens), recomenda-se o uso de scripts automatizados via GitHub CLI ou a integração com um serviço de armazenamento externo (como Firebase Storage) para evitar limitações de tamanho do repositório Git.
